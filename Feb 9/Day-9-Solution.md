# Day 9 Solution of 30 Days Of Nodejs By Scaler

**Question-**

```
Which of the following are examples of node modules?

a. Express

b. Body-parser

c. Socket.io

d. All of the above
```

**_Answer-_**
The answer to this question is **_d. All of the above_**.

```
Express js,Body-parser, Socket.io all are  example of node modules. They help in building web applications and real time communication.

1.Express.js- Express.js is a flexible Node.js web application framework that provides a set of features for building web and mobile applications. It is designed for building web applications and APIs.
2.Body-parser- Body-parser is a middleware for Express.js that parses incoming request bodies. When a client submits data to a server through an HTTP request the data is typically included in the body of the request. Body-parser parses this data and makes it available in the req.body property of the Express request object.
3.Socket.IO: Socket.IO is a JavaScript library that enables real-time, bidirectional communication between web clients (such as browsers) and servers.
```
